Idea general: 

Encontrar todas las apariciones de una palabra clave en un documento

Metodologia

1- chatGPT: arma un .txt al cual se le aplique la busqueda
2- implementar una lectura de archivos en python
3- guardar las palabras en un array
4- ordenar el array de palabras con el ordenamiento quicksort
5- implementar una busqueda binaria para determinar la existencia y posicion de una palabra
6- registrar el tiempo que tarda en hacerlo y compararlo con el metodo nativo que ofrece python
